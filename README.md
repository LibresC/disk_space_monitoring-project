# Disk Space Monitor

A PowerShell script that scans all local drives, calculates available disk space, and displays a warning when free space falls below a configurable threshold (15% by default).

## Features

- Checks all local fixed drives
- Calculates free space percentage
- Displays detailed disk usage information
- Generates warnings for low disk space
- Lightweight and easy to customize

## Usage

```powershell
.\CheckDiskSpace.ps1
